# Third-party notices

This game bundles third-party assets that ship under their own licenses. When you
add any of the assets below, keep this file accurate and include the upstream
license text where required.

## Fonts (`assets/fonts/`)

The game injects a CJK font at runtime (`ThemeManager` + `AssetLib.font()`), and
`export_presets.cfg` bundles `*.ttf` / `*.otf` into builds. If you drop one of the
recommended fonts in, you are redistributing it — include its license:

| Font | License | Where to get / license text |
|------|---------|------|
| Noto Sans CJK SC | SIL Open Font License 1.1 | https://github.com/notofonts/noto-cjk → `LICENSE` |
| Source Han Sans SC | SIL Open Font License 1.1 | https://github.com/adobe-fonts/source-han-sans → `LICENSE.txt` |

To comply with the OFL when shipping a font:

1. Put the font file in `assets/fonts/` (e.g. `assets/fonts/NotoSansSC-Regular.otf`).
2. Put its license next to it as `assets/fonts/<FontName>-OFL.txt`.
3. Do **not** rename the font's internal family in a way the OFL forbids (don't use
   "Noto" / "Source Han" in a *modified* font's reserved name).

The OFL is a free/libre license: redistribution inside an application (including
commercial) is permitted; the font itself may not be sold on its own.
