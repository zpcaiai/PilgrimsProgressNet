# Asset inventory & to-supply list

The project loads **every** asset through existence checks (`AssetLib`,
`AudioManager`, `ThemeManager`), so the game runs with any subset present and
greyboxes the rest. This file lists what exists, what's missing, and exactly
where to drop files so they light up — no code changes needed.

Legend: ✅ present · ⚠️ sparse · ❌ missing (wired, awaiting files)

---

## 1. Fonts — ❌ highest impact

| Need | Folder | Filename | Hook |
|------|--------|----------|------|
| CJK font | `assets/fonts/` | any `.ttf`/`.otf` (e.g. `main.ttf`) | `ThemeManager` → `AssetLib.font()` |

Without this, chat / player names / Chinese UI render as tofu (□). One Noto Sans
CJK or Source Han Sans Regular covers Latin+CJK. See `assets/fonts/README.md`.

## 2. Chat stickers (built-in packs) — ⚠️ placeholder art generated

18 placeholder PNGs are already generated by `tools/gen_sticker_placeholders.py`
(re-runnable). Replace any with final 128²/256² transparent art using the same
name from `assets/ui/stickers/manifest.json`:

- `assets/ui/stickers/pilgrim/` — `amen, pray, peace, press_on, take_heart, well_done, rejoice, keep_watch, do_not_fear, almost_home`
- `assets/ui/stickers/faces/` — `smile, laugh, cry, fear, hope, tired, grateful, think`

Sent as `sticker://pack/name`; rendered locally on every client. See
`assets/ui/stickers/README.md`.

## 3. Default avatar — ❌ optional (procedural fallback already works)

| Need | Folder | Filename | Hook |
|------|--------|----------|------|
| Fallback avatar | `assets/ui/` | `avatar_default.png` (square) | `AvatarUtil.default_texture()` |

If absent, a procedural circular silhouette (color seeded by name) is generated,
so blank avatars never appear. Supplying a PNG overrides the procedural one.
